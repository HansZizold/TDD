# TEST DRIVEN DEVELOPMENT

> In this project, a class 'Solver' with three procedures (factorial, reverse string and fizzbuzz) are implemented under the Test Driven Development (TDD) methodology.

## Getting Started

- To run this project, you need to install Ruby and run ```ruby main.rb```.
- To run the tests, you need to install rspec for Ruby and run ```rspec ./classes/spec```.
